Post-it Map

Map Poster | 03.2016

This poster is a map showing the whole Post-it Note functions in different areas. Because the main function of the Post-it Note is that they can links our thinking, I abstract logic thinking and emotional thinking: using the straight line represents logic thinking, and the curve line represents emotional thinking. The whole purpose of using Post-it Note can be devided into these two different ways. As a result, when you think of Post-it Note, you will find they are all in this map. This poster is a try to abstract the symbol to represent the meaning which I want to say and it is also a good test for me about map poster.
